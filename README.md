# teslaTrader
An app to predict trade-in value for all of your Teslas, using model/condition/milage/etc and an already constructed coreML model to predic pricings.

## Setup
App uses CoreML and already-setup model for learning

## Tech
1. Swift 4
2. iOS 11
3. Xcode 9
4. CoreML
